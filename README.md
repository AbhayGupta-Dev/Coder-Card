# Coder-Card
Coder Card is a Python GUI app I built to showcase my development skills and help fellow developers easily create stylish digital visiting cards. Designed with CustomTkinter, it lets you personalize your name, role, and contact—and save your card instantly with one click.


**🧑‍💻 Coder Card**

A sleek, customizable desktop app for developers to create and save personalized coder identity cards.
Built with Python and CustomTkinter, this tool blends modern UI design with practical functionality—perfect for showcasing your professional persona.

**✨ Features**

⚡ Live Preview → Instantly see your coder card update as you type.

🎨 Custom Branding → Add your name, role, and contact info with clean typography & digital aesthetics.

🖼️ Save as Image → Export your card as a .jpg file for use in portfolios, resumes, or social media.

🪄 Minimal UI → Designed with a focus on clarity, contrast, and developer-friendly visuals.

**🛠️ Tech Stack**

🐍 Python 3.10+

🖌️ CustomTkinter → Modern & elegant GUI styling

🖼️ PIL (Pillow) → Image rendering and export

**🚀 Getting Started**
1. Clone the repo
git clone https://github.com/AbhayGupta-Dev/Coder-Card.git
cd Coder-Card

2. Install dependencies
pip install customtkinter pillow

3. Run the app
python coder_card.py
